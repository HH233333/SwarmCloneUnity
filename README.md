# SwarmCloneUnity
这里是 SwarmClone 的直播画面合成项目。
## 技术实现方式
1. 为本项目接入live2D SDK
2. 利用Unity的UnityWebRequest异步发送请求到我们的AI服务器，并接收响应，根据AI返回的情感分析结果，更新Live2D模型的参数来改变动作和表情。
在没有接收到响应时，采用算法随机进行一些晃动让模型的表现更自然。
3. 后续步骤
